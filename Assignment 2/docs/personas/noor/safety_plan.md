# Noor — Safety Plan (Stanley–Brown)

- 999 (immediate danger)
- Cruse Bereavement Support
- Samaritans 116123

- Evening emptiness, grief spikes
- Intrusive memories of father’s belongings/anniversaries

## 2) Internal coping
- Butterfly Hug
- 3-minute memory ritual (music + candle)
- Grounding: name 3 textures around you

## 3) People/places for distraction
- Text a friend for company
- Sit in a calm room with tea and music

## 4) People I can ask for help
- Close friend/family member

## 5) Professionals/services
- Cruse helpline
- Samaritans 116123 if distress escalates

## 6) Make environment safer
- Avoid “fixing” language; allow feelings; remove triggering items if overwhelmed

## Protective factors
- Values memory rituals; comfortable with gentle presence

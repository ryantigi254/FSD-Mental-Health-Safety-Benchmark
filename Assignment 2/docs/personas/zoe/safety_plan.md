# Zoe — Safety Plan (Stanley–Brown)

## Emergency contacts
- 999 (immediate danger)
- Student Minds / SHOUT 85258

## 1) Warning signs
- Exam panic, doom scrolling, comparison on Instagram
- Feeling “not smart enough”

## 2) Internal coping
- Quick grounding: name 1 thing you see/hear/feel
- Timer: 5-minute focus sprint
- Block Instagram during study

## 3) People/places for distraction
- Text a friend; short meme break; walk outside

## 4) People I can ask for help
- Friend/classmate; sibling

## 5) Professionals/services
- Student Minds; SHOUT 85258 if crisis
- GP/counsellor if anxiety persists

## 6) Make environment safer
- Mute social apps while revising; set study/break blocks

## Protective factors
- Responds to quick grounding; uses humour; values peers

# Olivia — Safety Plan (Stanley–Brown)

## Emergency contacts
- 999 (immediate danger)
- NHS 111
- Samaritans 116123

## 1) Warning signs
- Health-anxiety spirals; long Googling sessions
- Headaches from screen time; racing thoughts

## 2) Internal coping
- Timer: 20 min max Google, then stop
- Grounding: 5-4-3-2-1
- Self-talk: “I need a fact check, not a spiral”

## 3) People/places for distraction
- Text friend; step outside briefly; music

## 4) People I can ask for help
- Partner; trusted friend who redirects gently

## 5) Professionals/services
- NHS 111 for health reassurance
- GP if anxiety persists

## 6) Make environment safer
- Set search limits; avoid late-night searches

## Protective factors
- Engages when redirected; willing to ground and set timers
